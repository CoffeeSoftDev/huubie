Cómo funciona el item activo del sidebar

El CSS define el item activo con border-left acento + degradado tenue de izquierda a derecha. El icono del item activo hereda --vsr-accent-soft aunque el formato del archivo tenga un color propio.

1. Lo que define el CSS

Reglas .sidebar-item y .sidebar-item.active en visor.css:473-486:

.sidebar-item {
    display: flex; align-items: center; gap: 9px;
    padding: 8px 14px;
    font-size: 12px;
    color: var(--vsr-text-mid);
    border-left: 2px solid transparent;
    transition: background-color .12s ease,
                color .12s ease;
}
.sidebar-item:hover {
    background: var(--vsr-bg-hover);
    color: var(--vsr-text);
}
.sidebar-item.active {
    background: linear-gradient(90deg,
        var(--vsr-accent-tint) 0%,
        transparent 100%);
    border-left-color: var(--vsr-accent);
    color: var(--vsr-text);
}
.sidebar-item.active .file-icon {
    color: var(--vsr-accent-soft);
}

2. Lo que entrega el render

Item active con pin y file-icon fmt-md:

mi-archivo.md ~3 KB

→ border-left 2px acento, degradado a la izquierda, icono accent-soft.
