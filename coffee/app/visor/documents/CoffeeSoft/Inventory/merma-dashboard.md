# Propuesta de dashboard - Mermas

**OBJETIVO**

Permitir un control detallado y visual de las mermas en la producción diaria de fogaza, para identificar y corregir rápidamente desviaciones fuera del margen aceptable y reducir pérdidas económicas.

**IMPACTO**

Si no se realiza a tiempo, las mermas seguirán fuera de un control, se incrementarán las pérdidas de costos y no se podrán tomar acciones correctivas oportunas en la producción.

**ACTIVIDADES**

1.  Recopilar datos históricos de mermas diarias de fogaza.
2.  Definir el margen aceptable de merma para cada producto.
3.  Diseñar una plantilla visual (dashboard o reporte gráfico) que destaque las mermas fuera del margen.
4.  Implementar el sistema de visualización en la plataforma de gestión interna.
5.  Capacitar al equipo de producción en el uso del sistema y el reporte de datos diarios.
