Cierre Diario Simple (MVP)

Implementacion minima viable. Un boton que ejecute el cierre diario con las validaciones basicas.

Alcance

  • ✓ Boton "Cerrar Dia" en el modal existente
  • ✓ Validacion de turnos abiertos
  • ✓ Consolidacion de metricas de turnos cerrados
  • ✓ Registro en daily_closure + tablas relacionadas
  • ✓ Bloqueo de creacion de pedidos post-cierre

Fases

1

Backend - Endpoint closeDailyShift

Archivo: ctrl-pedidos.php

Flujo:

  1. Recibir date / subsidiaries_id
  2. Validar no exista cierre previo
  3. Validar no haya turnos abiertos
  4. Calcular metricas consolidadas
  5. INSERT daily_closure
  6. INSERT closure_payment x3
  7. INSERT closure_status_proccess xN
  8. UPDATE order
  9. UPDATE cash_shift
  10. Retornar { status: 200 }

Queries nuevas:

  • getDailyConsolidatedMetrics()
  • getOpenShiftsByDate()
  • linkShiftsToClosure()

Queries reutilizadas:

  • getDailyClosureByDate()
  • createDailyClosure()
  • updateOrdersDailyClosure()
