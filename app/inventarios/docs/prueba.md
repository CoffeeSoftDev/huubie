# Lista de pendientes — Traspaso de productos

## Modal de nuevo inventario

- [x] En la barra de filtros de **categoría**, agregar un selector que diga **"Sin categoría"**. Esto permite agregar un traspaso sin categoría; al seleccionar una categoría, la tabla de *Buscar producto* solo muestra el filtro de esa categoría.
- [x] Al crear un traspaso y hacer clic en el botón **Crear y enviar traspaso**, debe preguntar si se desea confirmar la realización del traspaso.
- [ ] Revisar el agregado de **tipos de traspaso**.
