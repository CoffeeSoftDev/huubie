Acerca del Grimorio

¿Que es Huubie UI?

Huubie UI es el sistema de diseno oficial del producto Huubie. Define tokens, paleta dark, layout canonico y un set de componentes con clases .cs-* declaradas en css/ui-kit.css. Esta pagina es la entrada navegable al grimorio: el documento maestro vive en docs/grimorio-ui-kit.md.

Codigo del sistema

Todas las clases .cs-* viven en el CSS unico del kit.

css/ui-kit.css

Como usar Huubie UI en un template

  1. Importa el CSS: <link rel="stylesheet" href="../css/ui-kit.css">
  2. Carga Inter de Google Fonts y aplica font-family: 'Inter' al body.
  3. Body siempre: <body class="bg-[#111928] text-white">.
  4. Sigue el layout canonico (primaryLayout) como esqueleto.
  5. Prefiere clases .cs-* antes que utilidades Tailwind sueltas.
  6. NO mezcles con coffee-varoch (.cv-*) ni generes toggle light/dark.
