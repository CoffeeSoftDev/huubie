Cómo funciona el resaltado activo

El JS pide una pastilla terracota con icono blanco. El CSS del módulo la reinterpreta como item plano con border-left de acento y degradado tenue.

1. Lo que pide el JS

Plantilla original en sidebar.js:105-119:

<button class="w-12 h-12 bg-[#C05A40]
  rounded-xl flex items-center
  justify-center transition">
  <i data-lucide="house"
     class="w-6 h-6 text-white"></i>
</button>

→ pastilla terracota sólida con icono blanco.

2. Lo que entrega el CSS

Tras pasar por almacen-sidebar.css:49-55:

→ degradado tenue + border-left 3px + icono terracota.

Regla CSS que hace la magia

El selector #menu-sidebar > button.bg-\[\#C05A40\] detecta la clase que aplica el JS y la reinterpreta:

#menu-sidebar > button.bg-\[\#C05A40\] {
    background: linear-gradient(90deg,
        rgba(192, 90, 64, 0.18) 0%,
        transparent 100%) !important;
    border-left-color: #C05A40 !important;
}
#menu-sidebar > button.bg-\[\#C05A40\] i {
    color: #C05A40 !important;
}
